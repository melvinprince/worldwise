# Worldwise

## Overview

**Worldwise** is a React-based web application designed to provide users with insightful information about countries worldwide. Leveraging the power of Vite for rapid development and ESLint for code quality, this project offers a streamlined setup for building interactive user interfaces.

## Features

- **React Integration**: Utilizes React for building dynamic and responsive user interfaces.
- **Vite Configuration**: Employs Vite for fast development and optimized builds.
- **ESLint Support**: Integrates ESLint for maintaining code quality and consistency.

## Technologies Used

- **Frontend**: React.js
- **Build Tool**: Vite
- **Linting**: ESLint

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/melvinprince/worldwise.git
   cd worldwise
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Start the Development Server**:

   ```bash
   npm run dev
   ```

   The application will run on `http://localhost:3000`.

## Usage

1. **Explore Countries**: Navigate through the application to discover information about various countries.
2. **Search Functionality**: Use the search feature to find specific countries quickly.
3. **Responsive Design**: Access the application on different devices for a seamless experience.
